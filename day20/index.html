Linux

Linux is free, open-source operating systme. It is the most use OS in the world.

Linux Commands

pwd
Shows where you are in the file system
whoami
Shows what user you are operating as at the moment
ls
Lists all the files in the current directory
ls -l
Lists all the files in the current directory with full details
cd
Change directory
mv
Moves a file or directory to another locations. Also used for renaming files.
sudo
Execute the following command as the Super User aka:root
su
Execute the following command as the Super User aka:root
sudo su
Become the super user aka:root
man
Shows the user manual for the following command
unzip
Extract a zip file
apt-get
Used for installing software
ip address
Shows the mappings for the network interfaces. YOUR ip addres will be listed under #2 in the section labeled as "inet"
exit
Exit the current user's command prompt
sudo shutdown now
Shuts down the whole linux system.
history
Shows you the history of commands entered by the current user
more
Paginates the contents of the specified file or input stream
less
Creates a scrollable display for the specified file or input stream

Linux Command Line Shortcuts

[tab]
Auto-complete file or directory names
[up arrow]
Scroll backword through previous commands
[up down]
Scroll forward through previous commands
[q]
Quit a current explanation screen and return to the commmand prompt
[ctrl]+[c]
Abort the currently running command
